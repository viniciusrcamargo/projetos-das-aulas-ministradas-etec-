Introdução

Este é um exemplo de página HTML que usa tags semânticas para estruturar o conteúdo.

Tags semânticas

Tags semânticas são aquelas que dão um significado ao conteúdo, ao invés de apenas definir a aparência ou o comportamento. Por exemplo, a tag <header> indica que o conteúdo dentro dela é o cabeçalho da página, e a tag <nav> indica que o conteúdo dentro dela é a navegação do site.

Usar tags semânticas ajuda a melhorar a acessibilidade, a otimização para mecanismos de busca e a manutenção do código.
